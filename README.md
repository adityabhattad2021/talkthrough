# talkthrough

Practice real conversations in Indian languages — offline, free, powered by on-device AI.

Talkthrough drops you into everyday scenarios (negotiating an auto fare, ordering chai, talking to a doctor) and lets you roleplay them with an AI character running locally on your phone. No internet needed. No subscription. No data leaves your device.

**v0 languages:** Kannada ← Hindi

## How it works

1. **Pick a scenario** from the pathway
2. **Read the briefing** — cultural context + key vocabulary
3. **Warm up** — listen, repeat, get the words in your mouth
4. **Roleplay** — voice conversation with an on-device AI character
5. **Review** — get feedback on pronunciation, vocab, and cultural etiquette

## Stack

- React Native
- Gemma 4 (quantized, on-device via MediaPipe / llama.cpp)
- On-device STT & TTS
- SQLite for local persistence
- No backend
